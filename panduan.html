Mulai Panduan Ke Halaman Utama
Fase Pertama

Alur Dasar Pendaftaran

Ikuti langkah-langkah awal ini untuk memastikan data Anda tercatat dengan benar dalam sistem kami.

01

Pahami Syarat & Ketentuan

Langkah pertama pendaftaran adalah memahami Syarat & Ketentuan yang berlaku. Pastikan Anda telah membaca dokumen persyaratan secara lengkap.

Baca Persyaratan Lihat Ketentuan
02

Login Portal Peserta

Gunakan akun Single Sign-On (SSO) aktif Anda untuk masuk ke sistem pendaftaran MBKM USK Unggul.

Ke Halaman Login
03

Registrasi Program

Setelah login, akses menu Pendaftaran -> Registrasi Program. Lengkapi profil dasar Anda sebelum mulai memilih program.

04

Pilih Program MBKM

Sistem akan menyaring daftar program yang sesuai dengan Program Studi Anda. Klik rincian untuk melihat silabus dan syarat khusus.

Penting Untuk Diketahui

Persyaratan Kelompok MBKM

  • ✓ Maksimal 6 orang per tim.
  • ✓ Ketua tim wajib berstatus mahasiswa aktif.
  • ✓ Seluruh anggota wajib melakukan konfirmasi portal.
  • ✓ Proposal dikumpulkan dalam format PDF (maks. 5MB).
Unduh Format Proposal
Fase Kedua

Kategori Individu & Kelompok

Pilih jalur pendaftaran yang sesuai dengan rencana kegiatan MBKM Anda.

Kategori Individu

Jalur pendaftaran personal untuk mahasiswa yang ingin mengambil program secara mandiri.

01

Submit Pendaftaran

Pilih program yang diinginkan, lalu klik tombol Submit Pendaftaran untuk konfirmasi awal.

02

Input Ekivalensi

Masukan daftar mata kuliah yang akan dikonversi. Setelah selesai, klik Kembali ke Laman Berkas.

03

Finalisasi Data

Lengkapi berkas pendaftaran sesuai program. Bubuhkan tanda tangan digital pada kolom tersedia, lalu klik Finalisasi Pendaftaran.

Kategori Kelompok

Tingkatkan dampak kolaborasi dengan proyek berbasis tim yang terstruktur.

01

Pendaftaran oleh Ketua

Ketua kelompok melakukan submit pendaftaran program kategori kelompok, lalu mendaftarkan 5 orang anggota.

02

Proposal Kolektif

Unggah proposal kelompok (.pdf). Setelah itu, mintalah setiap anggota login untuk melakukan finalisasi berkas di akun masing-masing.

03

Submit Akhir

Ketua mengunci data tim setelah semua anggota menyetujui detail proyek di sistem.

Fase Akhir

Hasil Seleksi & Pasca Program

Cek halaman Pengumuman Seleksi. Jika dinyatakan lulus, Anda wajib mengunjungi menu Lengkapi Data untuk pengisian informasi lanjutan.
